git reset HEAD .

git branch  查看分支

git branch xxx  创建分支

git checkout xxx 检出到xxx分支

git status  查看工作区代码与暂存区代码的区别

git commit -m 'xxx' 将暂存区的代码提交到本地仓库 （sourcetree  提交按钮）

git add  将工作区的代码添加到暂存区 （sourcetree 文件前面的对勾按钮）

git push 本地版本库推送到远程服务器

git 三种状态

已提交（committed）
已修改（modified）
已暂存 (staged）
